# Godot场景创建指南

**项目**: 《镇魔司:缘起》原型  
**任务**: 创建玩家场景和测试关卡

---

## 🎮 步骤1：创建玩家场景（Player.tscn）

### 1.1 启动Godot并打开项目
```bash
~/bin/godot4 --path /home/fenrir/.openclaw/workspace-game-leaddev/zhenmosi-prototype
```

### 1.2 创建新场景
1. 点击菜单 **Scene → New Scene**
2. 选择 **Other Node** 类型
3. 搜索并选择 **CharacterBody2D**
4. 点击 **Create**

### 1.3 重命名根节点
- 将 `CharacterBody2D` 重命名为 `Player`

### 1.4 添加子节点

#### 添加Sprite2D（精灵图）
1. 右键 `Player` → **Add Child Node**
2. 搜索 `Sprite2D`，选择并创建
3. 在Inspector中，暂时不设置纹理（等待美术资源）

#### 添加CollisionShape2D（碰撞体）
1. 右键 `Player` → **Add Child Node**
2. 搜索 `CollisionShape2D`，选择并创建
3. 在Inspector中，点击 **Shape** → **New RectangleShape2D**
4. 调整大小：约64x64像素

#### 添加AnimationPlayer（动画播放器）
1. 右键 `Player` → **Add Child Node**
2. 搜索 `AnimationPlayer`，选择并创建

### 1.5 附加脚本
1. 选中 `Player` 根节点
2. 点击右侧 **Attach Script** 图标（链接图标）
3. **Script Path** 选择：`res://scripts/player/Player.gd`
4. 点击 **Load**（不要创建新脚本，加载已有的）

### 1.6 保存场景
- **Scene → Save Scene**
- 保存为：`res://scenes/player/Player.tscn`

✅ **玩家场景创建完成！**

---

## 🗺️ 步骤2：创建测试关卡（TestLevel.tscn）

### 2.1 创建新场景
1. 点击 **Scene → New Scene**
2. 选择 **Other Node**
3. 搜索并选择 **Node2D**
4. 点击 **Create**

### 2.2 重命名根节点
- 将 `Node2D` 重命名为 `TestLevel`

### 2.3 添加背景色（可选）
1. 右键 `TestLevel` → **Add Child Node**
2. 搜索 `ColorRect`
3. 在Inspector中：
   - **Size**: 1920x1080
   - **Color**: 深蓝色 (#1a1a2e)

### 2.4 添加平台

#### 创建第一个平台
1. 右键 `TestLevel` → **Add Child Node**
2. 搜索 `StaticBody2D`，创建
3. 重命名为 `Platform1`

#### 添加碰撞和视觉
1. 右键 `Platform1` → **Add Child Node** → `CollisionShape2D`
2. Shape选择 **New RectangleShape2D**
3. 调整大小：约400x50像素

4. 右键 `Platform1` → **Add Child Node** → `ColorRect`
5. 设置颜色为棕色 (#8b4513)
6. 调整大小与碰撞体一致

#### 创建更多平台
- 复制 `Platform1`（Ctrl+D）
- 移动到不同位置
- 创建3-5个平台用于测试跳跃

### 2.5 添加玩家实例
1. 右键 `TestLevel` → **Instantiate Child Scene**
2. 选择 `res://scenes/player/Player.tscn`
3. 将玩家放在第一个平台上方

### 2.6 添加相机
1. 右键 `Player` 实例 → **Add Child Node**
2. 搜索 `Camera2D`
3. 在Inspector中：
   - **Enabled**: ✅
   - **Zoom**: (2, 2) 或 (1, 1)
   - **Position Smoothing**: ✅
   - **Smoothing Speed**: 5.0

### 2.7 保存场景
- **Scene → Save Scene**
- 保存为：`res://scenes/test/TestLevel.tscn`

### 2.8 设置为主场景
- **Project → Project Settings → Application → Run**
- **Main Scene**: 选择 `res://scenes/test/TestLevel.tscn`

✅ **测试关卡创建完成！**

## ▶️ 步骤3：运行测试

### 3.1 运行游戏
- 点击右上角 **播放按钮** (▶️)
- 或按 **F5**

### 3.2 测试操作
- **A/D 或 方向键**：左右移动
- **空格 或 W**：跳跃
- **J 或 鼠标左键**：攻击（暂无效果）

### 3.3 预期效果
- ✅ 玩家能左右移动
- ✅ 玩家能跳跃
- ✅ 玩家受重力影响下落
- ✅ 玩家能站在平台上
- ✅ 相机跟随玩家

## 🐛 常见问题

### 问题1：玩家穿透平台
**解决**：
- 检查Player的碰撞层（Layer）和平台的碰撞层（Mask）
- 确保CollisionShape2D有正确的Shape

### 问题2：玩家不受重力影响
**解决**：
- 检查 `project.godot` 中的重力设置
- 确保Player脚本正确应用重力

### 问题3：移动速度太快/太慢
**调整**：
- 修改 `Player.gd` 中的 `SPEED` 常量
